# Instructies voor Claude

- **Alles gaat via het account van de gebruiker, niet via Claude.** Commits, PR's, releases en andere GitHub-activiteit worden uitsluitend op naam van de gebruiker gedaan:
  - GEEN `Co-Authored-By: Claude …`-regel in commit messages.
  - GEEN "🤖 Generated with [Claude Code]"-voettekst in PR-beschrijvingen, release notes of issues.
  - Ook geen andere Claude/AI-attributie of -branding in het repo, tenzij de gebruiker er expliciet om vraagt.
