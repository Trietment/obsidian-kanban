## Support the plugin

You can now support development of Trietment Kanban via Buy Me a Coffee.

- A funding link is shown on the plugin page, in the README, and at the bottom of the plugin's settings.
- This is entirely optional — the plugin stays free. Thank you for your support!

### Nederlands

## Steun de plugin

Je kunt de ontwikkeling van Trietment Kanban nu steunen via Buy Me a Coffee.

- Een funding-link staat op de plugin-pagina, in de README en onderaan de instellingen van de plugin.
- Dit is volledig optioneel — de plugin blijft gratis. Bedankt voor je steun!
